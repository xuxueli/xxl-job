# 基于quartz封装实现的的集群任务调度管理平台
	1、简单：支持通过Web页面对任务进行CRUD操作，操作简单，一分钟上手
	2、动态：支持动态修改任务状态，动态暂停/恢复任务，即时生效.
	3、集群：任务信息持久化到mysql中，支持Job服务器集群(高可用)，一个任务只会在其中一台服务器上执行
